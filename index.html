How It Works

A clear journey from signup to daily group operations

Every step is designed to feel simple for first-time users and clear enough for group leaders managing real money.

Create or join

Start a new group with a unique name or request to join an existing one.

Set your identity

Add your full name, phone number, and role so the group knows exactly who you are.

Run the chama

Open the dashboard to manage members, savings, loans, trust, chat, and education tips.

Choose your group type

Select SACCO, ROSCA, table banking, or a women-led group and continue with the right flow.

Check group name availability

New groups are protected from duplicates while existing groups can accept join requests by name.

Chairman approval

If details do not already match the group, the request is sent and waits for approval.

Daily money visibility

Once inside, members can follow contributions, available balance, trust, and discussion in one place.
