What this website is for

How To Start

If you are completely new

  1. 1. Open the website and click `Start Free` or `Create Your Group`.
  2. 2. Create your account or sign in if you already have one.
  3. 3. Choose whether you want to create a new group or join an existing one.
  4. 4. Enter the required group details and continue.
  5. 5. Fill in your full name, role, and phone number.
  6. 6. Open the dashboard and begin using the members, chat, activity, and tips sections.

Authentication

How sign up and sign in work

Sign up

A new user enters first name, last name, email, group name, and password. After registration, the site stores the user identity and uses it in later setup steps.

Sign in

A returning user enters email and password to open the platform again and continue from their existing group and profile setup.

Onboarding

Group setup

The user selects the type of group being created. This helps the platform choose the best workflow. The organisation name is also stored so the dashboard shows the real group identity.

Members

How members join the group

  1. 1. A new user clicks `Request to join` in the members section.
  2. 2. They enter full name, phone number, role, and referral.
  3. 3. The request goes into the pending join requests list.
  4. 4. The chairman reviews the request and chooses approve or reject.
  5. 5. When approved, the member moves into the main roster and member count updates.

Roles

How roles affect the group

  • `Chairman`: reviews join requests and oversees group actions.
  • `Treasurer`: focuses on contributions, funds, and balances.
  • `Secretary`: helps with records, organisation, and communication.
  • `Member`: participates in savings, loans, and group chat.

Chat

How the chat and bot work

Group communication

Members can communicate inside the chat area. Messages stay in chat history so users can come back and continue the conversation.

Bot control

The bot can be turned on or off. If on, it replies to balance, reminder, loan, and member questions. If off, members can still chat with each other.

Best Practice

Tips for operating the website well

  • Always finish onboarding and profile setup before expecting your personal name to show in the group roster.
  • Use the members section often so leadership and join activity stays current.
  • Review pending join requests before adding new financial access to a user.
  • Use chat history to keep group communication visible and organised.
  • Use the financial education section to train members who are new to the group.
